This template uses common features:

- Login with Token
- Password Resets
- Pusher Driver
- And more ...

## Postman Export

In this Template exists a archive JSON with all requests, but is a POSTMAN EXPORT.

## Install Project

### `composer install`
install dependecies
### `php artisan migrate --seed`
migrate and seed database
### `php artisan serve`
run dev server

### - Laravel Server in default port, 8000!
